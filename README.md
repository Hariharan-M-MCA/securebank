# SecureBank

SecureBank is a Spring Boot application (Java %JAVA_VERSION%) for demo banking features.

## Requirements
- Java 21
- Maven 3.6+

## Build
```powershell
cd C:\Users\Hariharan\IdeaProjects\securebank
mvn -B clean package
```

## Run
```powershell
mvn spring-boot:run
```

## Tests
```powershell
mvn -B test
```

## Notes
- This repository includes a GitHub Actions workflow that runs Maven build and tests on push/pull request.
